# weather
>基于vue2的天气查询小项目

# 技术栈

* vue2.0
* vuex
* vue-router
* webpack
* ES6
* Axios
* dayjs
* 


## Project setup
```
npm install
```

### 运行 Compiles and hot-reloads for development 
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
